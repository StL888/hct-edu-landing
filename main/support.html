FAQ

App startet nicht / Sicherheitswarnung
macOS: Beim ersten Start: Rechtsklick auf die App → «Öffnen». Ab v2.3 ist die App notarisiert — kein Warndialog mehr.
Windows: SmartScreen-Warnung → «Weitere Informationen» → «Trotzdem ausführen».
Wie richte ich den API-Key ein?
1. Gehe zu console.anthropic.com
2. Erstelle einen Account und lade API-Credits auf (ab $5)
3. Erstelle einen API-Key unter «API Keys»
4. In HCT: Einstellungen → SYSTEM → API-Key eingeben

Der Key beginnt mit sk-ant-... und wird sicher im System-Keychain gespeichert.
Analyse funktioniert nicht / Fehlermeldung
❶ API-Status prüfen: status.anthropic.com
❷ API-Guthaben prüfen: console.anthropic.com → Billing
❸ Key korrekt? Einstellungen → SYSTEM → Key prüfen (beginnt mit sk-ant-)

Häufigste Ursachen: Kein Guthaben, ungültiger Key, oder temporärer API-Ausfall.
Was kostet eine Analyse?
Analyse (Sonnet): ~$0,60–0,80 pro Report
Analyse (Opus Pro): ~$5,00 pro Report
PRIMER: ~$0,01–0,03 pro Nachricht
Follow-Up Check: ~$0,02

Die Kosten werden nach jeder Analyse und in der Sitzungsübersicht angezeigt.
Welches Modell soll ich nutzen?
Sonnet (Standard): Schnell, günstig, für Einsteiger und Fortgeschrittene ideal.
Opus Pro: Tiefere Analyse, nuancierter. Empfohlen für Experten-Level.

Opus Pro wird in den Einstellungen aktiviert und funktioniert am besten mit Level «Experte».
Werden meine Daten gespeichert?
Lokal: Alle Reports, PRIMER-Sessions und Einstellungen werden ausschließlich auf deinem Gerät gespeichert (SQLite-Datenbank).
API: Deine Analyse-Anfragen werden an Anthropic gesendet. Anthropic speichert keine API-Daten für Training (siehe Anthropic Privacy Policy).
Kein Tracking: HCT hat keine Analytics, kein Telemetrie, keine Server.
Auf welchen Plattformen läuft HCT?
macOS: Ab macOS 13 (Ventura). Intel + Apple Silicon. Notarisiert.
Windows: Ab Windows 10. Download als ZIP.
iPad: Native SwiftUI-App (iPadOS 17+). TestFlight-Beta.
Linux: Experimentell. Download als AppImage.
Was bedeutet der Status-Punkt in der Toolbar?
HCT zeigt den Live-Status der Anthropic API in der Toolbar an:

● grün — API erreichbar (All Systems Operational)
● orange — Eingeschränkt (Partially Degraded Service)
● rot — Nicht erreichbar (Major/Critical Outage)
● grau — Status unbekannt (kein Internet)

Klick auf den Punkt öffnet status.anthropic.com.

Fehlerbehebung:
1. Status prüfen: status.anthropic.com
2. Guthaben prüfen: console.anthropic.com → Billing
3. API-Key prüfen: Einstellungen → SYSTEM (beginnt mit sk-ant-)
